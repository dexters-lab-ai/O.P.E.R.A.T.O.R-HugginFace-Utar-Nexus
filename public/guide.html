User Guide

Welcome to O.P.E.R.A.T.O.R!

O.P.E.R.A.T.O.R (Optimized Protocol for Enhanced Robotic Automation Tasks & Online Responses) is your personal web assistant. It can help you automate tasks on websites, such as searching for products, filling out forms, or even performing blockchain operations like bridging or staking on Sonic Labs.

Watch Our Demo Video

This video walks you through the basics of using O.P.E.R.A.T.O.R, from running your first task to viewing your task history.

Getting Started

Follow these steps to start using O.P.E.R.A.T.O.R:

  • Log In: Use your email and password to access the dashboard.
  • Choose a Task Type: On the home page, select a task type from the Command Center (Manual, Repetitive, or Sonic Blockchain).
  • Enter a URL and Command: Provide the website URL and a natural language command (e.g., "Search for iPhone 14 on eBay").
  • Run the Task: Click the "Run" button and watch O.P.E.R.A.T.O.R execute your command.

Task Types

O.P.E.R.A.T.O.R supports different types of tasks to suit your needs:

  • Manual Task: Run a one-time task, like searching for a product or extracting data from a webpage.
  • Repetitive Task: Save a task to run multiple times (coming soon).
  • Sonic Blockchain: Perform blockchain operations like bridging assets or staking $S tokens on Sonic Labs.

Viewing Task Results

After a task completes, you can view the results in the Task Results section on the home page:

  • AI Results: A human-readable summary of what O.P.E.R.A.T.O.R did and the data it found.
  • Raw Results: The raw data extracted from the webpage, including screenshots and text.
  • Run Report: A detailed report of the task execution, including a replay of the actions taken.

Managing Your History

The History section lets you review past tasks:

  • Click on a history card to expand and see detailed results.
  • Use the "Rerun" button to repeat a task with the same URL and command.
  • Delete individual tasks or clear your entire history.

Settings

In the Settings page, you can:

  • Clear History: Remove all past tasks.
  • Set Custom URLs: Add default URLs that O.P.E.R.A.T.O.R can use for tasks (e.g., your favorite websites).

Tips for Better Commands

Here are some tips to get the best results from O.P.E.R.A.T.O.R:

  • Be specific: "Search for iPhone 14 Pro on eBay and extract prices" works better than "Find a phone."
  • Use natural language: Write commands as if you're giving instructions to a person.
  • Include the URL: Always provide the website URL to ensure O.P.E.R.A.T.O.R knows where to start.
